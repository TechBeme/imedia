    Create src/lib/link-analytics.ts with aggregation functions:
    - getLinkAnalytics(linkId: string, dateRange?: { from: Date; to: Date }): Promise<AnalyticsData>
      - totalClicks: count of all linkClicks for linkId
      - uniqueClicks: count of distinct IPs
      - clicksOverTime: array of { date: string; clicks: number } grouped by day
      - topCountries: array of { country: string; clicks: number } sorted by clicks desc, limit 10
      - devices: array of { device: string; clicks: number }
      - browsers: array of { browser: string; clicks: number }
      - operatingSystems: array of { os: string; clicks: number }
      - referrers: array of { referrer: string; clicks: number } sorted by clicks desc, limit 10
      - recentClicks: array of last 50 clicks with ip, country, device, browser, referrer, clickedAt
      - Use drizzle sql`` for aggregations (count, groupBy)

    Create src/app/api/links/[id]/analytics/route.ts:
    - GET handler: validate session, verify link ownership (link.userId === session.user.id)
    - Parse optional query params: from, to (ISO dates for date range filtering)
    - Call getLinkAnalytics(linkId, dateRange)
    - Return success({ analytics })
    - If link not found or not owned: return notFound() or unauthorized()
  </action>
