## Meta Graph API Reels Publishing

Reels use the same container + publish flow as images, but with:
- `media_type: "REELS"` in the container
- `video_url` instead of `image_url`
- Video must be under 60 seconds for Reels
- Cover image can be specified with `cover_url`

Endpoint: `POST /<IG_ID>/media`
Body: `{ media_type: "REELS", video_url: "...", caption: "...", cover_url: "..." }`

## Meta Graph API Stories Publishing

Stories use:
- `media_type: "STORIES"` in the container
- `image_url` or `video_url`
- No caption (Stories don't have captions via API)

Endpoint: `POST /<IG_ID>/media`
Body: `{ media_type: "STORIES", image_url: "..." }`
