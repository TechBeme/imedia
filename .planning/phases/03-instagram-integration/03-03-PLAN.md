## Meta Graph API Publishing Flow

1. Create media container: `POST /<IG_ID>/media`
   - Image: `{ image_url, caption }`
   - Carousel item: `{ image_url }` or `{ video_url, media_type: "VIDEO" }`
   - Returns: `{ id: container_id }`

2. For carousels, create carousel container: `POST /<IG_ID>/media`
   - `{ media_type: "CAROUSEL", children: "item1,item2,...", caption }`
   - Returns: `{ id: carousel_container_id }`

3. Publish container: `POST /<IG_ID>/media_publish`
   - `{ creation_id: container_id }`
   - Returns: `{ id: media_id }`

4. Check status: `GET /<CONTAINER_ID>?fields=status_code`
   - Status: `FINISHED`, `IN_PROGRESS`, `ERROR`

## Constraints
- Images must be JPEG
- Media must be on a publicly accessible URL at publish time
- Containers expire after 24 hours
- 100 posts per 24-hour rolling window
